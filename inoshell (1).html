Dokumentasi InoShell

Pengenalan

InoShell adalah terminal shell berbasis kode Java yang ringan dan fleksibel, dirancang untuk menjalankan perintah standar Linux dengan antarmuka pengguna minimalis.

Perintah Khusus

Kami memiliki perintah khusus untuk aplikasi kami sebagai berikut:

inosh-info            View information
inosh-sethome         Set HOME current path
inosh-setup-storage   Request storage permission

inosh -dc root        Get root status
inosh -ts <size>      Set text size
inosh -tc <color>     Set text color
inosh -tf <font>      Set text font

inosh -rls            List installed applications
inosh -uls            List user applications
inosh -ipkg <package> Remove an application
inosh -rm <package>   Remove an application
inosh -r <package>    Run an application

Penggunaan

InoShell dapat menjalankan perintah-perintah dasar Linux seperti:

ls       # Melihat daftar file dan folder
pwd      # Menampilkan direktori saat ini
cd       # Pindah direktori
mkdir    # Membuat folder baru
rm       # Menghapus file
cp       # Menyalin file
mv       # Memindahkan atau mengganti nama file
cat      # Menampilkan isi file
echo     # Menampilkan teks ke layar
